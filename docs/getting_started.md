#Getting started#

## Requirements ##
* Python 2.x >= 2.5
* Apache Libcloud
* Zfec

## Supported Python Versions ##
* Python from 2.5 up to 2.7

## Installation ##

### Using git ###
**System-Wide installation **

    $ git clone https://code.google.com/p/ulibcloud/
      $ cd ulibcloud
      $ python setup.py install

**Note** that you should have setuptools installed.
